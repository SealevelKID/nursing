💡 專案初心

本專案的誕生，是為了「讓政府公開資訊更便民」。
臺灣長照機構的官方原始資料往往分散、格式多變，且常因機構改名或更換負責人而產生資訊斷層。我們希望透過輕量、直覺的數位工具，協助家屬在為長輩做關鍵規劃時，能一鍵查詢行政裁罰、司法起訴與全網新聞，少走冤枉路。

📂 檔案架構與作用介紹

nursingindex.html
主網頁入口。建立乾淨、無廣告的卡片式佈局，並在頁尾貼心加入符合人性、不嚴肅的溫馨提示警語。

nursingstyle.css
網頁外觀外衣。專為「電腦版」與「手機版」雙平台進行文字大小與行高微調，確保家屬在戶外用手機查閱時依然清晰易讀。

nursingscript.js
核心前端大腦。負責動態渲染機構卡片，並搭載「智慧防金蟬脫殼」邏輯——自動掃描備註欄位中的改名軌跡，將新舊名稱、負責人、首支電話同步送出，交織成密不透風的 Google 查詢網。

data.json
結構化資料庫。存放清洗完畢後的長照機構基礎欄位，供前端即時調用，免去動態資料庫的沉重負擔。

process.py
後端清洗腳本。使用 Python 模組化結構，自動化串接、清洗並校對主管機關的原始數據，維持資料庫的純淨度。
